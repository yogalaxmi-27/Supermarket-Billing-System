# Supermarket Billing System (Console-Based Java)

A console-based Supermarket Billing System** developed using **Java**, designed to manage product billing operations with a **secure login system.
This application automates the billing process, calculates totals accurately, and reduces manual effort in supermarkets

# Features

- Login Authentication
-  Admin access control
-  Add products to bill
-  Generate customer bill
-  Automatic total calculation
-  Exit system securely
- Simple and efficient **console interface**

 Technologies Used

- Java (Core Java)
- OOP Concepts (Classes, Objects, Encapsulation)
- Exception Handling
- Loops & Conditional Statements
